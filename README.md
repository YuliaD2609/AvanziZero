**AvanziZero** è un'applicazione mobile sviluppata in Flutter pensata per semplificare la gestione della dispensa e combattere attivamente lo spreco alimentare. 

Nata per rispondere alle esigenze di organizzazione domestica di studenti fuorisede, coinquilini e famiglie, l'app mira a risolvere il problema degli acquisti ridondanti e dei prodotti dimenticati che superano la data di scadenza. Grazie all'integrazione di un sistema di intelligenza artificiale on-device, AvanziZero non si limita a un semplice inventario: impara dalle tue abitudini, automatizza l'inserimento della spesa tramite intelligenza artificiale e ti suggerisce come consumare al meglio ciò che hai già in casa limitando l'impatto ambientale.

## Funzionalità Principali

- **Dispensa Condivisa**: Tieni traccia degli alimenti e delle loro scadenze in tempo reale con i tuoi coinquilini.
- **Scansione Scontrini (OCR)**: Aggiungi rapidamente i prodotti scansionando lo scontrino. L'intelligenza artificiale riconosce e categorizza tutto in automatico.
- **Spesa Intelligente**: Lista della spesa condivisa e suggerimenti automatici basati sulle tue abitudini di consumo.
- **Ricette Anti-Spreco**: Suggerimenti di ricette mirate per utilizzare gli ingredienti in scadenza nella tua dispensa.
- **Privacy e Offline-First**: L'elaborazione AI avviene localmente sul dispositivo, garantendo privacy e il funzionamento base anche senza connessione.

## Compilazione e Avvio

Assicurati di avere [Flutter SDK](https://flutter.dev/) (versione 3.x) installato sul tuo sistema.

```bash
# 1. Clona il repository
git clone https://github.com/YuliaD2609/AvanziZero.git
cd AvanziZero/flutter_app

# 2. Scarica le dipendenze
flutter pub get

# 3. Avvia l'app (su un dispositivo fisico o emulatore)
flutter run
```
